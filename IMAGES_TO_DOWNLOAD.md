# Images à télécharger depuis les sites sources

## Sources
- **Eco Studio** : https://sites.google.com/view/cc-eco-studio/
- **Parking** : https://sites.google.com/view/cc-parking-space/

## Images nécessaires

### Logo C & C
- **Fichier** : `src/assets/img/logo-cc.jpg`
- **Description** : Logo "C & C" avec feuille entrelacée dans un cadre arrondi (foyer/sécurité)
- **Format recommandé** : 200x200px, JPG ou WebP

### Eco Studio - Images à récupérer depuis la galerie
- **Section** : Galerie / Gallery sur https://sites.google.com/view/cc-eco-studio/galerie-gallery
- **Images suggérées** :
  - Vue extérieure du studio
  - Vue intérieure (chambre avec lit double)
  - Cuisine équipée
  - Salle de bain
  - Terrasse
  - Vue sur les champs
  - Vue du village/forêt

### Parking - Images à récupérer depuis la galerie
- **Section** : Galerie / Gallery sur https://sites.google.com/view/cc-parking-space/galerie-gallery
- **Images suggérées** :
  - Vue de l'aire de stationnement
  - Terrasse avec vue
  - Prise électrique
  - Robinet extérieur
  - Vue générale du site

### Images d'emplacement
- **Section** : Emplacement / Location sur les deux sites
- **Images suggérées** :
  - Carte de localisation
  - Vue aérienne de La Sonnaz
  - Proximité autoroute A12
  - Gare Belfaux-Village

## Instructions pour télécharger

1. Visiter les sites sources Google Sites
2. Naviguer vers les sections "Galerie / Gallery"
3. Télécharger les images en haute résolution
4. Convertir en WebP pour optimiser (optionnel mais recommandé)
5. Placer dans `src/assets/img/`
6. Nommer les fichiers de manière descriptive :
   - `studio-exterieur.jpg`
   - `studio-interieur.jpg`
   - `studio-cuisine.jpg`
   - `parking-vue.jpg`
   - `parking-terrasse.jpg`
   - etc.

## Témoignages

Les témoignages sont disponibles sur :
- **Eco Studio** : Section "Témoignages / Testimonials"
- **Parking** : Section "Témoignages / Testimonials"

Les témoignages peuvent être ajoutés dans `src/_data/testimonials.json` après récupération.
